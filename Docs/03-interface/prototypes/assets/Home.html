Bem-vindo ao Sistema eSocial

Sistema para cadastro e envio de eventos eSocial referentes a processos trabalhistas. Gerencie os eventos S-2500, S-2501, S-2555 e S-3500 de forma simples e eficiente.

Processos Judiciais

Cadastro e consulta dos processos judiciais. Base para vinculação dos processos trabalhistas e geração dos eventos S-2500.

Processo Trabalhista (S-2500) Fase 1

Cadastro das informações do trabalhador vinculado ao processo judicial. Interface em formato wizard para facilitar o preenchimento.

Tributos (S-2501) Fase 1

Cadastro das informações de tributos decorrentes do processo trabalhista. Validação conforme tabelas do eSocial.

Consolidação (S-2555) Fase 2

Solicitação de consolidação das informações de tributos. Funcionalidade prevista para implementação futura.

Exclusões (S-3500) Fase 2

Exclusão de eventos enviados incorretamente. Permite cancelar eventos S-2500 ou S-2501 específicos.
